# Aegis Payroll A self-contained frontend MVP for a private payroll on-chain product. The app presents: - a polished landing page for the concept, - an interactive payroll batch demo, - selective-disclosure proof previews for employees, banks, auditors, and regulators, - architecture and impact sections for a hackathon pitch. ## Run No package manager or build step is required. 1. Open `index.html` in a browser. If you prefer serving the folder locally, any static file server will work. ## Files - `index.html` - app structure and content - `styles.css` - visual system, layout, responsive styling, motion - `script.js` - interactive demo state for payroll stages and proof generation - `favicon.svg` - local icon
